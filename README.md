# Advanced Number Guessing Game

A more advanced version of the classic number guessing game.

Players can choose between difficulty levels while earning points for successful guesses.

---

## Features

- Easy and Hard difficulty modes
- Score tracking
- Random number generation
- Distance-based hints
- Replay option

---

## Concepts Used

- Random Module
- Loops
- Conditional Statements
- Variables
- Score Systems
- User Input

---

## Why I Built This

After creating several basic guessing games, I wanted to combine everything I had learned into one larger project with multiple features.
